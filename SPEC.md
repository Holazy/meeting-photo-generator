# 腾讯会议合影生成器 - 规范文档

## 1. 项目概述

- **项目名称**: 腾讯会议合影生成器
- **项目类型**: 单页面Web应用
- **核心功能**: 生成多宫格会议合影，支持背景统一、姓名标注、人物抠图
- **目标用户**: 需要制作团队合影的企业/组织用户

## 2. UI/UX 规范

### 布局结构

- **整体布局**: 左侧控制面板（300px）+ 右侧预览区域（自适应）
- **控制面板**: 
  - 宫格选择区
  - 背景图片上传区
  - 导出按钮
- **预览区域**: 
  - 宫格画布展示
  - 每个宫格支持点击上传头像、编辑姓名

### 视觉设计

- **主色调**: 
  - 背景: #f5f7fa
  - 主色: #1890ff（腾讯会议蓝）
  - 强调色: #52c41a
  - 文字: #333333
  - 次要文字: #666666
- **字体**: 
  - 主字体: "Microsoft YaHei", "PingFang SC", sans-serif
  - 姓名字体大小: 14px
- **间距**: 
  - 面板内边距: 20px
  - 元素间距: 16px
- **效果**: 
  - 按钮hover: brightness(1.1)
  - 卡片阴影: 0 2px 8px rgba(0,0,0,0.1)
  - 过渡动画: 0.3s ease

### 组件设计

1. **宫格选择器**
   - 三个选项按钮：9宫格、20宫格、25宫格
   - 当前选中状态：蓝色边框 + 蓝色背景

2. **图片上传区**
   - 虚线边框拖拽区域
   - 支持点击上传
   - 预览缩略图

3. **宫格卡片**
   - 背景图片显示
   - 左下角黑色半透明矩形（姓名输入框）
   - 点击上传头像区域
   - 抠图后的人物头像居中显示

## 3. 功能规范

### 核心功能

1. **宫格分布选择**
   - 9宫格 (3x3)
   - 20宫格 (4x5 或 5x4)
   - 25宫格 (5x5)
   - 切换时清空当前数据

2. **背景图片上传**
   - 支持jpg、png格式
   - 自动适应所有宫格
   - 点击可重新上传

3. **姓名编辑**
   - 左下角黑色半透明矩形
   - 输入框可编辑姓名
   - 默认显示"参会人姓名"
   - 最多显示8个字符

4. **人物头像上传与抠图**
   - 支持点击上传头像
   - 使用 @imgly/background-removal 进行自动抠图
   - 抠图完成后显示透明背景人物
   - 支持重新上传覆盖

5. **导出功能**
   - 导出为PNG图片
   - 包含所有宫格内容
   - 点击按钮触发下载

### 技术实现

- 使用Canvas进行图片合成
- 使用 @imgly/background-removal 进行浏览器端抠图
- 纯前端实现，无需后端

## 4. 验收标准

- [ ] 可以选择9/20/25宫格
- [ ] 上传背景图片后所有宫格显示同一背景
- [ ] 每个宫格可以编辑姓名
- [ ] 每个宫格可以上传人物头像
- [ ] 人物头像自动抠图（去除背景）
- [ ] 可以导出合并后的完整图片
- [ ] 界面美观，操作流畅
